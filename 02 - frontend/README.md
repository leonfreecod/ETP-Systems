# Frontend - ETP Systems

Esta pasta contém o **Front-end do sistema ETP Systems**.

O front-end é responsável pela **interface do usuário**, permitindo que os colaboradores interajam com a plataforma, acessem cursos, acompanhem seu progresso e utilizem as funcionalidades do sistema.

O front-end se comunica com o **Back-end através de APIs**, que fornecem os dados necessários para a aplicação funcionar.

---

## 🎯 Objetivo

Criar uma interface **simples, intuitiva e acessível**, permitindo que os usuários utilizem a plataforma de forma fácil para acessar conteúdos e acompanhar sua evolução profissional.

---

## 🖥️ Principais funcionalidades

- Tela de **login e autenticação**
- **Cadastro de usuários**
- **Dashboard** com visão geral dos cursos
- Listagem de **cursos disponíveis**
- Visualização de **cursos em andamento**
- Acompanhamento do **progresso do usuário**

---

## ⚙️ Tecnologias previstas

O projeto pode utilizar uma das seguintes tecnologias:

- **React**
ou
- **Flutter Web**

Essas tecnologias serão responsáveis por construir a **interface da aplicação** e realizar requisições para a API do sistema.

### Descrição das pastas

**components**  
Componentes reutilizáveis da interface.

**pages**  
Páginas principais da aplicação.

**services**  
Responsáveis pela comunicação com a API do Back-end.

**assets**  
Imagens, ícones e outros recursos visuais.

**styles**  
Arquivos de estilo e layout da aplicação.

---

## 🔗 Comunicação com o Back-end

O Front-end irá consumir a **API do sistema**, responsável por fornecer dados como:

- usuários
- cursos
- progresso de aprendizado
- autenticação

---

## 🚀 Objetivo desta camada

Fornecer uma **experiência de uso clara e eficiente**, permitindo que os usuários naveguem pela plataforma **ETP Systems** de forma simples e intuitiva.
